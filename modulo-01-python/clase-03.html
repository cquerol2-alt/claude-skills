📖 Resumen de conceptos

Condicionales: if, elif, else

En Python, las condicionales se basan en indentación, no en llaves como en JavaScript:

edad = 18

if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres adolescente")
else:
    print("Eres menor")

💡 Diferencia clave con JS: No hay llaves {}, la indentación es obligatoria y parte de la sintaxis.

Bucles: for y while

Bucle for: Itera sobre secuencias (listas, tuplas, strings, rangos).

# Iterar sobre un rango
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# Iterar sobre una lista
colores = ["rojo", "verde", "azul"]
for color in colores:
    print(color)

# Con enumerate() para obtener índice
for idx, color in enumerate(colores):
    print(f"{idx}: {color}")

Bucle while: Ejecuta mientras se cumpla la condición.

contador = 0
while contador < 5:
    print(contador)
    contador += 1

range(): generador de secuencias

range() crea una secuencia inmutable de números:

range(5)           # 0, 1, 2, 3, 4
range(1, 6)        # 1, 2, 3, 4, 5
range(0, 10, 2)    # 0, 2, 4, 6, 8 (paso 2)

enumerate(): índice + valor

Obtén tanto el índice como el valor sin usar variables adicionales:

estudiantes = ["Ana", "Bruno", "Clara"]
for i, nombre in enumerate(estudiantes):
    print(f"{i+1}. {nombre}")
# 1. Ana
# 2. Bruno
# 3. Clara

List Comprehensions: crear listas de forma elegante

Construye listas en una sola línea. ¡Esto NO existe en JavaScript!

# Básica: elevar al cuadrado
numeros = [1, 2, 3, 4, 5]
cuadrados = [x**2 for x in numeros]
# [1, 4, 9, 16, 25]

# Con condición: solo pares
pares = [x for x in numeros if x % 2 == 0]
# [2, 4]

# Transformación + condición
resultado = [x*2 for x in numeros if x > 2]
# [6, 8, 10]

Dict Comprehensions: crear diccionarios

Mismo concepto que list comprehensions, pero para diccionarios:

numeros = [1, 2, 3, 4, 5]
# {número: cuadrado}
cuadrados_dict = {x: x**2 for x in numeros}
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# Con condición: solo pares
pares_dict = {x: x**2 for x in numeros if x % 2 == 0}
# {2: 4, 4: 16}

Comparativa Python vs JavaScript

Concepto Python JavaScript
Sintaxis Indentación obligatoria Llaves {}
range() range(1, 6) Array.from({length: 5}, (_, i) => i+1)
Comprensiones ✅ [x**2 for x in lista] ❌ No existen nativamente
Alternativa en JS — lista.map(x => x**2)
enumerate() ✅ for i, v in enumerate(lista) ❌ for (let [i, v] of lista.entries())

💻 Práctica guiada

Ejercicio 1: Validar edad con condicionales

Escribe una función que valide la edad y devuelva un mensaje:

def validar_edad(edad):
    if edad < 0:
        return "Edad inválida"
    elif edad < 13:
        return "Eres un niño"
    elif edad < 18:
        return "Eres adolescente"
    else:
        return "Eres adulto"

print(validar_edad(25))  # "Eres adulto"

Ejercicio 2: Bucle for con range()

Imprime la tabla de multiplicar del 3:

numero = 3
for i in range(1, 11):
    print(f"{numero} x {i} = {numero * i}")
# 3 x 1 = 3
# 3 x 2 = 6
# ...

Ejercicio 3: List comprehension con filtro

Extrae solo los números mayores de 10 y dóblalos:

numeros = [5, 12, 8, 15, 20, 3, 18]
resultado = [x * 2 for x in numeros if x > 10]
print(resultado)  # [24, 30, 40, 36]

Ejercicio 4: Dict comprehension con enumerate()

Crea un diccionario con posición como clave y valor al cuadrado:

valores = [2, 4, 6, 8]
resultado = {i: v**2 for i, v in enumerate(valores)}
print(resultado)  # {0: 4, 1: 16, 2: 36, 3: 64}

Ejercicio 5: Bucle while con contador

Suma números del 1 al 10:

suma = 0
i = 1
while i <= 10:
    suma += i
    i += 1
print(f"La suma es: {suma}")  # 55

🧪 Mini-reto: FizzBuzz con comprensiones

El clásico FizzBuzz, versión Python elegante

Reglas:

  • Números del 1 al 100
  • Si múltiplo de 3 → "Fizz"
  • Si múltiplo de 5 → "Buzz"
  • Si múltiplo de ambos → "FizzBuzz"
  • Si no → el número

Solución con list comprehension:

fizzbuzz = [
    "FizzBuzz" if i % 15 == 0 else
    "Fizz" if i % 3 == 0 else
    "Buzz" if i % 5 == 0 else
    i
    for i in range(1, 101)
]

# Imprime los primeros 20
for item in fizzbuzz[:20]:
    print(item)

Solución con bucle tradicional:

for i in range(1, 101):
    if i % 15 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)

💡 Ambas funcionan. Elige la que te parezca más legible. En producción, la versión con bucle es más clara para otros desarrolladores.
